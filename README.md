# 📋 COIG Przypomnienie 

Rozszerzenie Brave/Chrome przypominające o wystawieniu wniosku o pracę zdalną w systemie COIG.

## Wymagania

- Przeglądarka `Brave` / `Google Chrome`
- Dostęp do systemu pracownika `COIG`

## Instalacja

1. Pobierz lub sklonuj repozytorium
2. Otwórz przeglądarkę i wejdź na

    - `chrome://extensions/`
    - `brave://extensions/`
  
4. Włącz **Tryb dewelopera** (prawy górny róg)
5. Kliknij **Załaduj rozpakowane** i wskaż folder z rozszerzeniem

## Funkcje

- Przypomnienie w wybrany dzień tygodnia o wybranej godzinie
- Powiadomienie zawiera informację, na które dni należy wypełnić wniosek
- Naprzemienne tygodnie: **Standardowy** i **Z piątkiem**
- Możliwość ręcznego ustawienia typu następnego tygodnia
- Automatyczne przełączanie między typami po każdym powiadomieniu
- Przycisk szybkiego przejścia do formularza COIG

Po otrzymaniu powiadomienia kliknij je lub przycisk „Otwórz formularz", aby przejść bezpośrednio do systemu COIG.
